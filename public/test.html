QuickBase Voice Messages

Send a test voice message.

Use this form to send a test message. The text of the message will be read from a text field in an existing QuickBase record. The message will be sent using your Twilio account. Twilio will convert the text to an audio message and play it to the recipient at their phone number.

Except for 'QuickBase Realm' and 'QuickBase Application Token', the fields below are all required.

If your QuickBase table has a 'voice_mail_error_message' application variable, the recipient will receive the text in that variable if an error occurs while playing the message.

Please enter numbers only in the Record Number, Field ID and Phone fields.
