QuickBase Voice Messages

Send bulk voice messages.

Use this form to send bulk voice messages.

The messages will be sent using your Twilio account.

You will also need a SimpleWorker account so that the messages can be sent in a background job.

The QuickBase Application must have these application variables:

  1. simpleworker_access_key
  2. simpleworker_secret_key
  3. twilio_account_sid
  4. twilio_auth_token
  5. twilio_outbound_phone_number

The QuickBase Table must have a 'Pending Voice Messages' report. The report must contain these fields:

  1. Record ID#
  2. Phone
  3. Message
  4. Sent

Except for 'QuickBase Realm' and 'QuickBase Application Token', the fields below are all required.
